Privacy Policy for Link Revealer

Effective Date: September 2025

Overview

Link Revealer is designed to improve browsing safety by expanding shortened or obfuscated URLs and showing users the true destination before they click. This extension does not collect, transmit, or sell personal data.

Permissions Used

  • activeTab – Only runs when the user clicks the extension icon. Used to expand the current tab’s links.
  • scripting – Used to inject content scripts and CSS into the current tab after user action.
  • storage – Saves simple preferences like tooltip style. No personal data is stored.

Data Processing

  • No personal or sensitive data is collected, stored, or transmitted.
  • No analytics or tracking is performed.
  • All link expansion and processing happen locally in your browser.
